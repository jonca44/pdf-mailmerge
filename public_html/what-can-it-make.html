What Can It Make?

Mail Merge can be used to create almost any paperwork. This includes making things suchs a reminder notices, tickets, invitiations, greeting cards, mailing labels, invoices, invoices and much more. Any paperwork you have that you need to add data to and print for many people can be created as a mail merge.



Rocket Mail Merge is a great choice for creating :

  • Mailing Labels
  • Reminder Notices
  • Invitations
  • Appointment Confirmations
  • Letters
  • Invoices
  • Greeting Cards
  • Personalised Advertising
  • Bulk Mail
  • Coupons
  • Event Tickets
  • ... and many other types of paperwork in bulk!
